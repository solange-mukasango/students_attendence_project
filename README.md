 step1 :First-time user registration

On the home page, find the “Register” or “Sign Up” link.

Click it to open the registration form.

Fill in the required information:

Username or Full Name

Email (if applicable)

Password

Other fields (like class, role, etc., if your project has them)

Click Register / Create Account.

This will save your account in the database.

Step 2: Login

Go back to the Login page.

Enter the username/email and password you just registered.

Click Login.

If successful, you will be redirected to the dashboard/home page of the attendance system.

Step 3: Start using the system

After logging in, the user can:

Add students

Take attendance

View attendance reports

Update or delete student records

Tip: If your system supports roles (like Admin or Teacher), the first registered user can be set as Admin to manage all features.
